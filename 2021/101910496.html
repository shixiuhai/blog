(2) 实现客户网络课用

​ 实现客户网络恢复正常， 二层上网的方式参考此配置（这里的二层配置的是接口对模式，实际上还有交换模式可以选择配置方式和接口对类似）,三层可上网方式参考此配置，实际在安装部署防火墙的时候，因为是导入的配置文件，我们只需要在配置文件的基础上对防火墙端口的修改或者增减就可以了。

2. 电光口的使用

实际业务部署中会出现使用光口还是使用电口的方式

## 进如命令模式进行命令配置
system-view 

## 从使用电口切换到使用光口
int 接口(例:int g0/0/1)
combo enable fiber

## 从使用光口切换到使用电口
int 接口(例：int g0/0/1)
combo-port copper

3. IPSEC修改

(1) 在网络接口处修改或者创建loopback地址

如果是导入的配置文件一般都是已经存在了loopback地址，我们只需要进行一下修改即可

(2) 配置IPSEC本端端口

  • 第一种知道IPSEC 预共享密码 的情况下

  • 第二种不知道 预共享密码 的情况下

    一般需要通过命令把本端接口修改为其他端口，因为在图形界面下修改本端端口 预共享密码 会消失，在不知道密码的情况下我们是无法进行配置的

    (1) 查看默认配置文件下的本端端口

    我们得走默认本端端口是GE1/0/1

    (2) 命令方式修改本段端口

    ```

    进入端口

    int g0/0/1

    查看端口下的配置

    display this

    我们可以看到如下内容

    interface GigabitEthernet1/0/1
    undo shutdown
    ip address 192.168.137.6 255.255.255.0
    service-manage ping permit
    ipsec policy ipsec2010125724 # ipsec2010125724这个值是不断改变的，基于不同的防火墙。

    禁用g1/0/1做为IPSEC本端端口

    undo ipsec policy ipsec2010125724

    进入到我们需要修改的本段端口，比如g1/0/2

    进入端口

    int g1/0/2

    添加命令

    ipsec policy ipsec2010125724

    再次查看图形界面发现本端端口已经修改为g1/0/2这个端口了
